# 5001_KeyInputGame

**キーボードだけで完結するゲーム** — マウス／パッドを使わず、キーボード入力のみで操作が完結する
グリッド型リアルタイムアクション（タイピングゲームではない）。

- コア制約: 操作はキーボードのみ
- コアの面白さ: 正確・素早いキー入力そのものの快感
- 参照イメージ: ロックマンエグゼ／One Step From Eden
- エンジン: Godot 4 ／ 対象: PC（Steam 等）／ 開発元: beernosusume.net
