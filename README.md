# 5001_KeyInputGame

**キーボードだけで完結するゲーム** — マウス／パッドを使わず、キーボード入力のみで操作が完結する
（タイピングゲームではない）。グリッド上の **ATB タクティクス**＋**リアルタイム音ゲー式パリィ**＋**スレスパ式の分岐探索**。

- コア制約: 操作はキーボードのみ
- コアの面白さ: 正確・素早いキー入力の快感（音ゲー式パリィ＋好機のスキル）
- 参照イメージ: FF（ATB）／FFタクティクス／音ゲー式パリィ／Slay the Spire
- エンジン: Godot 4 ／ 対象: PC（Steam 等）／ 開発元: beernosusume.net
