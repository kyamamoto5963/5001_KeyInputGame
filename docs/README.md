# docs 索引（5001_KeyInputGame）

最終更新: 2026-06-18

## 前提・方針（確定事項）

- **コア制約**: 操作は**キーボードのみ**で完結。マウス・ゲームパッド前提の操作を入れない。
- **コアの面白さ**: 正確・素早いキー入力の快感。主にリアルタイムの**音ゲー式パリィ**と、ATBの好機に**スキルを的確に当てる**ことで出す。
- **ジャンル**: グリッド上の **ATB タクティクス**（FFのタイムバー）＋ **リアルタイム音ゲー式パリィ** ＋ **スレスパ式の分岐探索**。
- **参照イメージ**: FF（ATB）／FFタクティクス（グリッド移動）／音ゲー式パリィ（タイミング防御）／Slay the Spire（分岐マップ）。
- **対象プラットフォーム**: PC（ダウンロード／Steam 等を想定）。
- **スコープ**: ステージ／レベル制（分岐探索）で拡張。まずは縦スライス1本（1戦闘＋最小マップ）を遊べる所まで。
- **開発元**: beernosusume.net。
- **エンジン**: **Godot 4**（→ 05_tech-stack.md, ADR-001）。

## 用語集（命名レジストリ）

決めた呼び名はブレさせない。コード識別子・UI 文言・会話で同じ語を使う。

| 日本語（正） | コード識別子（英） | 意味 |
|---|---|---|
| グリッド | `grid` | 戦闘フィールド全体。**3×10 の共有フィールド**（味方・敵が同じ盤を共有、左右分けなし）。 |
| マス | `cell` | グリッドの1区画。味方・敵はマス単位で位置を持つ。 |
| パーティ | `party` | 出撃する味方の編成（**最大3人**）。 |
| メンバー | `member` | パーティの各キャラ。固有の移動範囲・ステータスを持つ。 |
| ATB | `atb` | タイムバー。**止まらず常時進行**（ポーズ除く）。溜まる速さは**ステータス＋スキル＋装備**依存。 |
| ウェイト | `wait` | ATB蓄積中の状態（指示待ちではない）。表記は「**ウェイト**」に統一（ウエイトは使わない）。なる瞬間＝ウェイト化。 |
| アクティブ | `active` | ATB満了し指示待ちの状態。なる瞬間＝アクティブ化。複数同時満了は内部管理順に並ぶ。 |
| フォーカス | `focus` | 現在操作中（指示入力中）のアクティブキャラ。**溜まった順**に自動でフォーカスが回る。 |
| ターン | `turn` | アクティブキャラを操作できる行動機会。移動（移動範囲内）＋スキル発動。 |
| 移動範囲 | `move_range` | 1ターンに動けるマス数。**メンバーごとに決まる**。 |
| 編成画面 | `loadout` | 出撃前にスキルを a/s/d/f キーへ割り当てる画面。 |
| スキル | `skill` | 攻撃手段。a/s/d/f に4つ割り当て。**クールタイム**と**攻撃種別**を持つ。**通常攻撃もスキル**（特別扱いしない）。 |
| 行動終了（パス） | `end_turn` | アクティブ中に `w` でターンを終える。スキル未使用なら次ATBは20%から再開。 |
| クールタイム | `cooltime` | スキルを再使用できるまでの待ち時間（「クールダウン」とは呼ばない）。 |
| マナ | `mana` | スキルのコスト資源。スキルはマナ＋クールタイムの両方を消費。 |
| 詠唱 | `cast` | スキル発動前の溜め。`詠唱中` 状態を経る（割り込みに弱い）。0＝即時。 |
| 地点攻撃 | `area_skill` | マス（地点）を対象にする攻撃種別。 |
| ターゲット攻撃 | `target_skill` | 特定の敵を対象にする攻撃種別。 |
| 向き | `facing` | キャラの向き。**左右のみ**（上下なし、絵は左右反転）。基本はスキル方向に自動、手動は q→左右。 |
| 状態異常 | `status` | 毒（継続ダメージ）・混乱（行動が乱れる）・スタン 等。付与確率・持続を持つ。 |
| 床生成（フィールド） | `field` | スキルでマス属性を生成（ダメージ床等）。範囲・持続を持つ。 |
| パリィ | `parry` | 音ゲー式の防御。**独立リアルタイム**で常時可能（攻撃/スキルの2種）。難度は**シビア寄りで調整**。 |
| パリィリング | `parry_ring` | パリィ時に縮むわっか。円に重なる瞬間に押すとノーダメージ。 |
| 攻撃パリィ / スキルパリィ | `parry_attack` / `parry_skill` | 通常攻撃／スキル攻撃に対するパリィ。**r=攻撃パリィ / e=スキルパリィ**。 |
| 敵 | `enemy` | 倒す対象。雑魚もボスも、1体のことも複数のことも。 |
| 雑魚 / ボス | `mob` / `boss` | 通常敵／強敵。 |
| 戦闘 | `battle` | 1ノードの戦い。多対多になりうる。 |
| 探索マップ | `map` | スレスパ式の**分岐**で奥へ進む全体マップ。 |
| ノード | `node` | マップ上の到達地点（戦闘・ボス・その他）。 |
| ステージ | `stage` | レベル/章の単位。クリアで次へ。 |
| ワールドマップ / エリアマップ | `world_map` / `area_map` | 2層のマップ。エリア端→別エリア。1作目はワールドマップ開放まで。 |
| ポイント / ルート | `point` / `route` | マップ上の地点／地点を結ぶ道。 |
| 中間ポイント | `mid_point` | ルート上に点在。イベント・戦闘が起きる。 |
| 拠点 | `base_point` | 町・村・王都など拠点的な点。メニュー選択式（自由移動でなく行ける場所がメニュー）。 |
| 探索ポイント | `explore_point` | 草原・ダンジョン等。中に入って探索（基本全探索）。 |
| ユニークスキル | `unique_skill` | 教会の祈りでガチャ割り当て。入手済みはプールから選択可。 |

> 未確定の呼称・追加要素が出たらここに追記して正典化する。世界観・進行の構想は [08_world-vision.md](08_world-vision.md)。
