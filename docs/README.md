## 前提・方針（確定事項）

- **コア制約**: 操作は**キーボードのみ**で完結。マウス・ゲームパッド前提の操作を入れない。
- **コアの面白さ**: 正確・素早いキー入力そのものの快感。
- **参照イメージ**: ロックマンエグゼ（グリッド移動＋技）／One Step From Eden（グリッド型リアルタイムアクション）。
- **対象プラットフォーム**: PC（ダウンロード／Steam 等を想定）。
- **スコープ**: ステージ／レベル制で拡張していく。まずは縦スライス1本を遊べる所まで。
- **開発元**: beernosusume.net。
- **エンジン**: **Godot 4**（→ 05_tech-stack.md, ADR-001）。

## 用語集（命名レジストリ）

決めた呼び名はブレさせない。コード識別子・UI 文言・会話で同じ語を使う。

| 日本語（正） | コード識別子（英） | 意味 |
|---|---|---|
| グリッド | `grid` | 戦闘フィールド全体。マスの集合。 |
| マス | `cell` | グリッドの1区画。プレイヤー・敵はマス単位で位置を持つ。 |
| プレイヤー | `player` | 操作キャラ。自陣エリア内をマス移動する。 |
| 自陣 / 敵陣 | `player_side` / `enemy_side` | グリッドのプレイヤー側／敵側エリア（エグゼ型に左右で分割）。 |
| 技 | `move` | プレイヤーの攻撃手段。**固定セット**を各キーに割り当てて使いこなす。 |
| 敵 | `enemy` | 倒す対象。複数の雑敵が出る。 |
| ウェーブ | `wave` | 一度に出現する敵のまとまり。 |
| ステージ | `stage` | レベル単位。クリアで次へ。 |
| クールダウン | `cooldown` | 技を再使用できるまでの待ち時間。 |

> 未確定の呼称・追加要素が出たらここに追記して正典化する。
