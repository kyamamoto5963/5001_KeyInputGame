# 02 コアループ・コンセプト ★契約レイヤ

最終更新: 2026-06-18
状態: ドラフト

> 全要素はこのコアループに奉仕する。盛る前に、まずこれを縦スライスで成立させる。

## コアループ（1〜2行）

**グリッド上をマス移動で素早く動き回り、固定の技を正確・素早いキー入力で出して、
押し寄せる複数の雑敵をさばく。** これを 1 ステージ＝数十秒〜数分の単位で繰り返す。

## 1分の体験（プレイヤーが繰り返して面白いこと）

1. 敵の動き・弾を見る（読む）
2. 安全なマスへ**正確に・素早く**移動する（方向キー）
3. 好機に**技キー**を正確に叩いて敵を削る
4. これを連続でテンポよく決められると気持ちいい ← ここが快感の核

## キーボードだけで完結する操作（コア制約）

- **移動**: 方向キー or WASD でマス単位移動（自陣エリア内）。
- **技**: 各技を個別キーに割り当て（例: J/K/L など）。**固定セット**を手で覚えて使いこなす。
- **システム操作**（開始・決定・戻る・ポーズ）も全部キーで完結させる。
- ※具体キー割り当ては 06_systems.md（入力定義）で確定。リマップ可否は要確認。

## こだわり演出を1つ（候補・要決定）

シンプル基調を保ちつつ、本質を表す演出を1つだけ仕込む。候補:

- **入力に呼応する手応え演出**: 技ヒット時の短いヒットストップ＋効果音＋画面の軽い反応で、
  「正確に速く押せた」瞬間を快感として返す。コア（入力の快感）と直結するのでこれを第一候補とする。

> ★印（契約レイヤ）。小さく保つ。コアが面白くなってからジュース（演出）を足す。

## 縦スライスの目標（まずここまで）

タイトル → 1ステージ（グリッドで雑敵のウェーブをさばく）→ クリア/失敗 → リザルト → 再挑戦/タイトル、
が**一気通貫で回り、触って「入力が気持ちいい」と感じる**状態。ここで面白さを検証し、ダメなら方向転換する。

## 既定値（縦スライスの仮置き・調整前提）

- グリッド: 横6×縦3、プレイヤーは左の自陣3×3を移動（エグゼ型）。← 要確認・調整
- 技: 3〜4個の固定技から開始（射撃系＋範囲系など）。← 要確認
- 敵: 敵陣に雑敵が複数、ウェーブで出現。全滅でクリア。← 要確認
