# 02 コアループ・コンセプト ★契約レイヤ

最終更新: 2026-06-22
状態: ドラフト（ATB タクティクス＋音ゲーパリィ＋分岐探索に更新）

> 全要素はこのコアループに奉仕する。盛る前に、まずこれを縦スライスで成立させる。
> 戦闘の詳細仕様（盤面・状態遷移・強制移動）は [11_battle-spec.md](11_battle-spec.md)。

## コアループ（1〜2行）

**3×10 の共有グリッドで、ATB が溜まったメンバーを移動＋スキルで的確に動かしつつ、
敵の攻撃には独立リアルタイムの音ゲー式パリィでタイミングよく応じて、多対多の戦闘を捌く。**
勝ったらスレスパ式の分岐マップで次のノードを選び、奥へ進む。

## 1分の体験（プレイヤーが繰り返して面白いこと）

1. ATB バーが時間で溜まる（**リアルタイムに進行、止まらない**）。敵は満了で自動行動、味方は満了で**アクティブ**になり指示待ち。
2. アクティブになったメンバーへ**溜まった順にフォーカス**。**移動とスキルの両方**を行える。入力は **移動 → スキルキー → ターゲット** の順を**コマンドスタック方式**で積む（格ゲーのプラクティス式 → [11_battle-spec.md](11_battle-spec.md) §2.5）。
   移動歩数ぶん（1キー=1歩）**位置取り**し、a/s/d/f の**スキル**を1つ選び、対象（地点 or 敵）を決めて撃つ（クールタイムを見て選ぶ）。
   ※**早く指示しないと損**（待機中も時間は流れ、敵は攻撃する）。
3. **並行して**、敵の攻撃が飛んできたら r/e で**音ゲー式パリィ**（縮むわっかが円に合う瞬間に押下→ノーダメージ）。← ここがリアルタイムの緊張＝快感の核。
4. 多対多を捌いて勝利 → 分岐マップで次ノードを選び前進。これを繰り返す。

> 本作の肝は**全部リアルタイム**で重なること: **複数のアクティブ味方を素早く捌く × 飛来攻撃をパリィする**。
> 時間は止まらない（ポーズ除く）ので、パリィは操作中・待機中いつでも割り込んで成立する。詳細は [11_battle-spec.md](11_battle-spec.md) §1.5。

## キーボードだけで完結する操作（コア制約）

- **移動（右手）**: 方向キー or jikl or hjkl でマス移動。**1キー=1歩**、1ターンに積めるのは**移動歩数**まで（メンバー固有・`→←→`=3歩）。
- **スキル（左手）**: **a / s / d / f** に4スキル（編成画面で割り当て）。地点攻撃はマス、ターゲット攻撃は敵を対象。
- **パリィ（左手）**: **r=攻撃パリィ / e=スキルパリィ**。独立リアルタイム。判定は**シビア寄り**で調整。
- **ターン入力の順**: 移動 → スキルキー → ターゲット。
- **システム操作**（開始・決定・戻る・ポーズ）も全部キーで完結させる。
- ※移動キー候補（方向/jikl/hjkl）の確定・対象選択方式は [06_systems.md](06_systems.md) で詰める。リマップ可否は要確認。

## アニメーション優先（パリィの見た目）

- 他アクション（移動・スキル）中に攻撃が来たら、**そのアクションのアニメを優先**（パリィ判定は内部で成立）。
- ウェイト中（手が空いている）に攻撃が来たら、**パリィモーション**を再生する。

## こだわり演出を1つ（候補・要決定）

シンプル基調を保ちつつ、本質を表す演出を1つだけ仕込む。第一候補:

- **パリィ成功の手応え演出**: ジャスト押下で短いヒットストップ＋効果音＋わっかが弾ける反応。
  コア（正確・素早い入力の快感）と直結するのでこれを第一候補とする。コアが面白くなってから足す。

## 縦スライスの目標（まずここまで）

タイトル → 編成（スキルを a/s/d/f に割り当て）→ 最小の分岐マップ → **1戦闘**（ATB＋パリィで多対多を捌く）→
クリア/失敗 → リザルト → 再挑戦/タイトル、が**一気通貫で回り、触って「パリィとスキルが気持ちいい」と感じる**状態。
ここで面白さを検証し、ダメなら方向転換する。

## テンポ方針

- 目標は **LoL くらいの中速・タクティカル**。One Step from Eden は速すぎと感じる方向。速くしすぎない。
- 敵攻撃は**着弾約1秒前**にパリィナビ（縮むわっか）が出る＝これがテンポの自然なブレーキ（詳細 [11_battle-spec.md](11_battle-spec.md) §7）。

## 既定値（縦スライスの仮置き・調整前提）

- グリッド: 3×10 共有フィールド（味方左寄り・敵右寄りで開始など、初期配置は要確認）。
- パーティ: 2〜3人から開始。移動歩数はメンバーで差をつける（例: 2〜4歩）。← 要確認
- スキル: 各メンバー a/s/d/f に4つ。地点/ターゲットとクールタイムを混ぜる。← 要確認
- 敵: 雑魚2〜3体の1戦闘から。ボスは縦スライス後。← 要確認
- パリィ: まず攻撃パリィ(r)で判定を作り、スキルパリィ(e)を足す。判定幅はシビア寄りから入れて調整。← 要確認
